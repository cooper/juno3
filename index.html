the third version of juno-ircd, an IRC daemon written in Perl

juno3 is the third version of juno-ircd, an extremely customizable and fully-featured modular IRCd. Because this version is a completely rewrite, it boasts major code improvements as well as more possibilities for the extremely useful module API. Unlike previous versions of juno, this version is able to link to itself and create a single-network feel among several IRC servers.

Dependencies

None (all core Perl modules) Net::DNS and its dependencies if you wish to enable hostname resolving IO::Socket::IP or IO::Socket::INET6 for IPv6 support IO::Socket::SSL for SSL support

Install

This IRCd comes ready. Just create your configuration and save it to etc/ircd.conf and start the IRCd with ./juno start.
